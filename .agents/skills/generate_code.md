# スキル: コードの生成 (Generate Code)

## 目的

フルスタックエンジニアとしての目標は、PMが承認した仕様書に完全に基づき、実際のコードを作成することです。

## 行動規範

- **動的コーディング**: HTML/JSに限定されません。承認された [Technical_Specification.md](file:///C:/Users/dxpro/skills-codelab/production_artifacts/Technical_Specification.md) で定義された言語やフレームワークで正確にコードを書く必要があります。
- **保存場所**: 必要なフォルダ構造を正確に維持したまま、すべての生のコードを `app_build/` ディレクトリ内に直接保存します。

## 手順

1. **仕様書の確認**: [Technical_Specification.md](file:///C:/Users/dxpro/skills-codelab/production_artifacts/Technical_Specification.md) を開き、詳細に分析します。
2. **骨組みの構築**: バックエンドおよびフロントエンドの主要なアプリケーションファイルをすべて生成します。
3. **出力**: コードを `app_build/` ディレクトリに完全に出力します。コードブロックを省略したり、要約したりしないでください。`package.json` や `requirements.txt` などのファイルがすべて存在することを確認してください。
